Start Using GraniteDS Librairies, API Documentation, Tools and Sources

  • Libraries

    All GraniteDS librairies (JAR or SWC) are in the <JBOSS_HOME>/graniteds/librairies directory.

    You may use them for your new projects, targeted to this Community Platform or any other supported application server and technologies.

  • API documentation

    Go to the <JBOSS_HOME>/graniteds/docs directory and unzip the API doc bundles (ActionScript3 or Java).

  • Tools

    Go to the <JBOSS_HOME>/graniteds/tools directory. Copy the org.granite.builder_***.jar file to your <ECLIPSE_HOME>/plugins directory.

    Restart Eclipse.

  • Sources

    GraniteDS sources are available in a separated package. Download it on GitHub here.
